## Recommended Learning Paths

### Path A: New to AI Coding (2-3 weeks)

```
Week 1:
├── Intro to Claude AI (Coursera) - 4 hours
├── Install Claude Code
└── Complete our ONBOARDING_CHECKLIST.md

Week 2:
├── Interactive Prompt Engineering Tutorial - 4 hours
├── Watch "How I Use Claude Code" (Builder.io)
└── Practice on real tasks

Week 3:
├── Claude Code course (Coursera) - 15 hours
└── Capture first learnings with /learn
```

### Path B: Experienced Developer (1-2 weeks)

```
Week 1:
├── Skim Official Docs (2 hours)
├── Read Best Practices blog post (30 min)
├── Complete ONBOARDING_CHECKLIST.md
└── Set up MCP servers

Week 2:
├── Building with Claude API (Coursera) - dive deep
├── OpenSSF Security Guide
└── Start contributing to team patterns
```

### Path C: Team Lead / Architect (1 week)

```
Day 1-2:
├── Official Best Practices (30 min)
├── AWS Deployment Patterns (1 hour)
└── OWASP AI Security Overview (1 hour)

Day 3-5:
├── Building with Claude API (selected modules)
├── MCP architecture deep dive
└── Review USAGE_POLICY.md and METRICS.md
```
